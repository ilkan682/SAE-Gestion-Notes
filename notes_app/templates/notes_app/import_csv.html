Import des notes par CSV

Format attendu du fichier CSV :
La première ligne doit contenir exactement cet en-tête :

numero_etudiant,id_examen,note,appreciation

Exemple : 
numero_etudiant,id_examen,note,appreciation
20240001,1,15.5,Bien
20240002,1,12.0,Assez bien
20240003,1,8.5,Insuffisant
  • numero_etudiant : le N° étudiant tel qu'il est en base
  • id_examen : l'ID numérique de l'examen
  • note : nombre entre 0 et 20 (ex: 15.5)
  • appreciation : texte libre, peut être vide
